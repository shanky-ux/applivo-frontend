Build a stunning, animated, highly interactive landing page for "Applivo" 
— an AI-powered job application automation desktop app for Windows.

Applivo runs as a local .exe file. It scrapes LinkedIn, Indeed, Internshala 
& Wellfound, scores jobs with GPT-4o AI, auto-generates tailored resumes 
& cover letters, and submits applications via Playwright browser automation 
— all running silently on the user's own PC.

🎨 DESIGN SYSTEM — LIGHT THEME
═══════════════════════════════════════════

Background: Pure white (#ffffff) with soft grey sections (#f8fafc)
Primary color: Vivid blue (#2563eb)
Secondary: Indigo-violet (#6366f1)
Accent: Emerald green (#10b981) for success states
Gradient: Blue → Indigo (used on CTAs, headings, highlights)

Cards: White with soft shadow + blue border on hover
  box-shadow: 0 4px 24px rgba(37,99,235,0.08)

ANIMATIONS (Framer Motion — all required):
- Hero elements: staggered fade-up on load (title → subtitle → buttons → terminal)
- Scroll-triggered: every section fades + slides up when entering viewport
- Stats: animated count-up from 0 when scrolled into view
- Cards: lift + blue glow on hover (translateY -6px + shadow)
- Buttons: scale pulse on hover + ripple click effect
- Terminal: real typewriter effect cycling through log lines
- How It Works: step connector line draws/fills as user scrolls
- FAQ: smooth spring accordion open/close
- Navbar: frosted glass blur appears on scroll (transparent → white/blur)
- Floating background: subtle animated gradient blobs (very light blue/indigo, 
  low opacity 0.06) slowly drifting behind hero — NOT distracting

9. FAQ
──────────────────────────────────────────
White background.
Title: "Frequently Asked Questions"

Accordion style — smooth spring animation on open/close.
Only one open at a time. Open item: blue left border + 
blue question text + chevron rotates 180°.

Q1: Is Applivo completely free?
A: Yes — free and open source under MIT License. You only 
   pay for your own OpenAI API usage (typically $2–5/month 
   for active job searching).

Q2: Do I need coding knowledge?
A: None at all. Download the .exe, fill in the setup form 
   with your API key and preferences, and click Start Agent.

Q3: Is it safe to enter my LinkedIn password?
A: Yes. Credentials are encrypted with AES-256-GCM using 
   a locally-derived key and stored only on your machine. 
   Nothing is sent to any server except LinkedIn/Indeed 
   and OpenAI directly.

Q4: Will LinkedIn detect and block me?
A: Applivo uses randomised delays, realistic browser 
   headers, and Playwright — mimicking real human browsing. 
   Running from your home IP makes detection very unlikely.

Q5: Can I control which jobs it applies to?
A: Yes. Set a minimum match score (e.g. 80%), a daily 
   application limit, and optionally require Telegram 
   approval before every single submission.

Q6: What job platforms does it support?
A: LinkedIn, Indeed, Internshala, and Wellfound — with 
   Glassdoor support coming soon.
